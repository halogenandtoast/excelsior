h1. Excelsior

Ruby gem that uses C bindings to read CSV files superfast (my benchmarks put this at 7.4 times faster than fastercsv). I'm totally serial!

h2. Usage

For right now all it can do is parse a text stream...

<pre>
require 'rubygems'
require 'excelsior'
rows = []
Excelsior::Reader.rows(File.new("a_csv_file.csv")) do |row|
 rows << row
end
</pre>
