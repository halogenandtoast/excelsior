h1. Excelsior

Ruby gem that uses C bindings to read CSV files superfast (my benchmarks put this at 7.4 times faster than fastercsv). I'm totally serial!

h2. Usage

h3. Parsing an IO Object

<pre>
require 'rubygems'
require 'excelsior'
rows = []
Excelsior::Reader.rows(File.new("a_csv_file.csv")) do |row|
 rows << row
end
</pre>

h3. Parsing a text stream

<pre>
require 'rubygems'
require 'excelsior'
rows = []
Excelsior::Reader.rows("first,second,third\n1,2,3") do |row|
 rows << row
end
</pre>
